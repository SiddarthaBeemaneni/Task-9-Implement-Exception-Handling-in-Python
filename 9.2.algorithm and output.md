Scenario 2: Age Validation for Voting

Create a Python program to check if the entered age is valid for voting. Handle cases where the user enters negative age or non-numeric input.

Algorithm:

1. Input age.

2. If input is not a number, raise exception "Invalid Input".

3. If age < 0, raise exception "Age cannot be negative!".

4. If age < 18, display "Not eligible to vote."

5. Else, display "Eligible to vote."

Output:

Enter your age: 19

You are eligible to vote.
